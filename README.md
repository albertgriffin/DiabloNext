# DiabloNext

DiabloNext is a **personal fork and gameplay/engine experiment** based on [DevilutionX](https://github.com/diasurgical/devilutionX).

This repository is no longer a stock DevilutionX mirror. The direction here is focused on substantial gameplay and engine-side changes, and features will diverge significantly from upstream behavior.

## Project Status

- Personal project (active experimentation)
- Will include breaking changes, unfinished systems, and balance overhauls
- Compatibility and behavior will differ from official DevilutionX releases

If you are looking for the stable, community-maintained upstream project, use:
- [DevilutionX repository](https://github.com/diasurgical/devilutionX)
- [DevilutionX wiki/manual](https://github.com/diasurgical/devilutionX/wiki)
- [DevilutionX releases](https://github.com/diasurgical/devilutionX/releases/latest)
